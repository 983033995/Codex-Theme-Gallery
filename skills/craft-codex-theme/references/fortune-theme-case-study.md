# 财神主题案例复盘

## 目标

将 Codex 打造成米白宣纸、红金财神、喜庆但可长期使用的生产界面。主题必须在真实 Codex 首页、历史对话、代码文件、设置页和侧边栏中工作。

## 关键迭代

1. **背景阶段**：最初只加入财神背景，白色标题、代码文字和设置项在亮色区域失去对比度。修正为独立表面层与深棕文字 tokens。
2. **侧边栏阶段**：统一普通操作图标为红金视觉，并为 hover、active、focus 和 pointer cursor 建立清晰反馈。
3. **项目图标阶段**：文件夹 SVG 在不同状态路径不一致，改为读取项目行 `aria-expanded`。收起使用钱袋，展开使用喷金元宝钱袋并带轻量弹出动画。
4. **首页阶段**：自定义 Dashboard 早于侧边栏图标挂载，首次克隆失败后图标永久为空。增加幂等 hydration，每次 Mutation/Resize 都补齐缺失图标；品牌标识使用独立资源。
5. **立体主视觉阶段**：财神头部越出卡片边界，背景、羊皮纸、人物切图分别控制层级，避免整张图被矩形裁剪。
6. **聚宝盆阶段**：装饰曾多次覆盖右侧文件/输出区。最终以“显示/隐藏侧边栏”按钮的 `aria-pressed` 为权威状态：右栏展开隐藏，收起显示；旧版才回退到贴右边缘的几何检测。
7. **持久化阶段**：主题只注入当前会话会在重启后消失，最终由菜单栏 App 接管 Provider、登录项、修复与切换。

## 必测矩阵

| 区域 | 状态 |
| --- | --- |
| 首页 | 首次进入、Mutation 后、窗口缩放 |
| 项目 | 收起、展开、切换动画 |
| 侧边栏 | hover、active、focus、长标题 |
| 历史任务 | 右栏展开、右栏收起 |
| 内容区 | 代码、文件树、diff、输出 |
| 设置 | 表单、说明文字、禁用控件 |
| 动态 | GIF 正常、reduced-motion 降级 |
| 生命周期 | Codex 重启、系统登录启动 |

## 不要重复的错误

- 只换背景，不重新设计文字与表面层。
- 用 DOM 路径或随机 class 判断状态。
- 让前景装饰通过左移继续覆盖正文。
- 首页图标只尝试挂载一次。
- 把 Emoji、CSS 图形或临时 SVG 当最终资产。
- 只验证首页，不验证历史任务、代码和设置。
